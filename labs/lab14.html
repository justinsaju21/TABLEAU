Objective: Iterate on a published dashboard, add Ship Mode filter and worksheet annotation.
📝 Tasks
Follow the step-by-step instructions below. Complete every task in order, save your Tableau workbook as a .twb file, then upload it to the XML Evaluator on the right. The evaluator reads your workbook’s XML structure — not screenshots.
🎯 Submission Checklist
Before uploading: open your .twb in VS Code and verify the required XML nodes are present. The evaluator checks attribute values — not the visual output.
Lab Instructions
1
Open the workbook you published in Exp 13. Add a Color encoding to one of your charts — for example, drag Ship Mode to the Color card on a bar chart. This immediately changes the visual and updates the XML
node.
✓ What you should seeThe bar chart marks are coloured differently based on the Ship Mode assigned.
2
Add a Ship Mode filter: drag Ship Mode to the Filters shelf. Right-click the filter card on the sheet > Apply to Worksheets > All Using This Data Source. Show the filter control. In your dashboard, the filter card now appears and affects all sheets.
✓ What you should seeA Ship Mode filter control appears on the dashboard and affects all connected sheets.
3
On a scatter plot or bar chart, right-click a specific data point and choose Annotate > Point. In the annotation text box type something like "Highest profit customer in Q3 — VIP retention target". Move the annotation so it does not overlap other marks.
✓ What you should seeA custom text annotation appears pointing to the specific data point you selected.
4
Create a brand-new Dashboard sheet. Rename it "Version Log". Drag a Text object onto it. In the text, document your changes: Version 1.0 (original), Version 1.1 (added color), Version 1.2 (Ship Mode filter), Version 1.3 (annotation). This is your changelog.
✓ What you should seeA new dashboard named "Version Log" appears containing your typed changelog text.
5
Go to Server > Tableau Public > Save to Tableau Public As and save the updated workbook. The URL will change slightly. Copy the new URL. Save locally as "Exp14_Versioning.twb" and upload here.
✓ What you should seeThe updated workbook is published to Tableau Public with a new/updated URL.
